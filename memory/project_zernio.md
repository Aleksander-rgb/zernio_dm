# Zernio — platforma social media management

Zernio to platforma do zarządzania mediami społecznościowymi obsługująca 14 platform.

**Why:** Projekt skupia się na integracji z Zernio API i automatyzacji działań social media przez Claude/AI.
**How to apply:** Przy zadaniach związanych z postami, reklamami, kontaktami — korzystaj z toolsów MCP opisanych w reference_zernio_mcp.md.

## Zasoby

- Dashboard: https://zernio.com
- API Keys: https://zernio.com/dashboard/api-keys
- Dokumentacja: https://docs.zernio.com
- Python SDK: https://github.com/zernio-dev/zernio-python
- MCP endpoint: https://mcp.zernio.com/mcp

## SDK

- Python: `pip install zernio-sdk` lub `pip install zernio-sdk[mcp]`
- Obsługa MCP przez pakiet `zernio-sdk[mcp]`
